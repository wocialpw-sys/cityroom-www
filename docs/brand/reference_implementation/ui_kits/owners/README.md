# Cityroom — Owners UI Kit

Marketing website recreation targeting apartment owners (właściciele mieszkań).

This is a **recreation**, not a new design — since no production site or Figma was provided, the kit follows the visual foundations documented in `/README.md` exactly: white-dominant, typography-led, one gradient moment per view, Ibrand-PL for display, Manrope for body, brand illustrations for feature blocks, Lucide icons.

**Files**
- `index.html` — entry, mounts the full landing page with all components composed together. Navigation links scroll to sections.
- `Nav.jsx` — sticky glass-blur top navigation.
- `Hero.jsx` — headline + lead + CTAs + hero illustration.
- `Stats.jsx` — "trust" stats band with gradient numbers.
- `HowItWorks.jsx` — 4-step process with Punktor bullets.
- `Services.jsx` — service feature grid using brand illustrations.
- `Testimonial.jsx` — owner quote section.
- `ContactForm.jsx` — "Umów rozmowę" contact form.
- `Footer.jsx` — footer with legal info.

All components are intentionally cosmetic — they render, they do not implement real form submission.
